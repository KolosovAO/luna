# Luna Helper

PWA-версия умеет ставиться на экран iPhone и создавать календарь с напоминаниями о молодом серпе. Можно выбрать дневное напоминание или вечернее напоминание для приметы с деньгами.

## Calendar reminders

На iPhone: открыть `https://kolosovao.github.io/luna/` в Safari, добавить на экран Домой, выбрать режим и нажать `Добавить в календарь`.

Кнопка скачивает `.ics` файл на ближайшие 18 лунных циклов. Его нужно открыть и добавить в календарь. Сервер, Cloudflare и Apple Developer аккаунт не нужны.

Локально можно проверить сборку так:

```bash
npm run build:pages
```
